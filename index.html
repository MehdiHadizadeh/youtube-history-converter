How to get your data:

  1. Go to takeout.google.com
  2. Deselect all, then select only "YouTube and YouTube Music"
  3. Click "Next step" → Choose file format and delivery method
  4. Wait for the export (you'll receive an email)
  5. Download and extract the ZIP file
  6. Find the "watch-history.html" file in the YouTube folder
  7. Upload it here to convert to JSON
